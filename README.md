# Brew16

Brew16 is a 16-bit, 286 protected mode, experimental hobby operating system written in Pascal and Assembly.

## Dependencies

`fpc i8086 cross-compiler`
`open-watcom-v2`
`nasm`

`make`
`git`
`qemu`

`coreutils`
`mtools`
(build FDA image)

## Build

```bash
make
```

## Run

```bash
make run
```

## Clean

```bash
make clean
```
